https://github.com/elixir-lang/elixir/issues/7989#issuecomment-408002011

## How to reproduce it

1. `mix deps.get`
2. `mix dialyzer`

## Elixir & Erlang/OTP versions

1.7.0-otp-21 & 21.0 both installed with asdf version manager (see `.tool-versions`)
